# Code of Conduct

This project expects respectful, inclusive, and constructive collaboration.

## Expected behavior

- Be considerate when reviewing or discussing work.
- Give actionable feedback focused on the code and project goals.
- Respect differing experience levels and communication styles.
- Avoid harassment, insults, discriminatory language, or personal attacks.

## Reporting

For a hackathon or classroom setting, report concerns to the project maintainer, mentor, or event organizer. Maintainers should review concerns promptly and take proportionate action to keep the project space safe and productive.

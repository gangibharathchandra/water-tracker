# Skill Documentation - Water Issue Tracker

## Project Name
Water Issue Tracker - Civic Complaint Management System


## Overview
Water Issue Tracker is a Streamlit based civic application that allows citizens to report water related problems and helps administrators track and resolve complaints.

The system provides a simple digital platform for communication between citizens and authorities.
